# AgriIntel

## Overview

Crop Intelligence System is a machine learning-based agricultural decision support system that identifies climate-optimal crops and estimates yield and economic losses caused by suboptimal crop choices.

## Problem Statement

Are farmers growing the most suitable crops for their climate conditions?

## Features

* Crop recommendation using Machine Learning
* Yield prediction using LightGBM
* Explainable AI using SHAP
* Economic opportunity loss analysis
* Climate suitability assessment

## Dataset
(Maharashtra, Uttar Pradesh & Punjab)
* Crop Recommendation Dataset
* Crop Production India Dataset
* Agmarknet Agricultural Price Dataset

## Models Used

* Decision Tree
* Random Forest
* XGBoost
* Stacking Ensemble
* LightGBM

## Key Results
* Random Forest Accuracy: 99.3%
* SHAP identified Humidity as the strongest predictor
*  ₹1,791 Crore estimated annual opportunity loss
*  100% crop-climate mismatch across analyzed states

## Technologies

Python, Pandas, NumPy, Scikit-Learn, XGBoost, LightGBM, SHAP, Google Colab
